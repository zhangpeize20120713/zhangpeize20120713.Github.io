Pixel 5 刷机与 Root

发表于 2023-08-15 更新于 2023-08-16 分类于 教程 ， Android 阅读次数： 本文字数： 942 阅读时长 ≈ 3 分钟
通过电脑给 Google Pixel 5 进行线刷与使用 Magisk Root 设备

开篇提醒

如果你没有 Root 请记得退出你的 Google 账户

解 BootLoader

在“设置” – “关于手机”里，连续点几次版本号来打开开发者模式，然后在“系统” – “开发者选项”中找到名为 OEM 解锁的选项，并将它开启

或者使用 ADB 命令解锁

然后通过音量键选择选项，电源键确认解锁
完成后输入

即可重启
重启完后进入“设置” – “系统” – “开发者选项”中找到名为 USB 调试的选项，并将它开启
最后将手机连接到电脑，当手机弹出 USB 调试请求的时候选择“确认”即可

特别提示：解 BootLoader会导致数据丢失，请先备份好数据再解锁

线刷

降级需清除数据刷机，否则可能导致丢失 IMEI 之类的问题

从 Google Play services 下载镜像，找到合适的版本，并选择 Link

如果你已配置好 ADB 与驱动可以直接下一步

从 developers 中下载驱动与 ADB 工具（Android SDK Platform-Tools）

完成后将 usb_driver 与 platform-tools 解压到空白文件夹
打开 usb_driver 文件夹，右键 android_winusb.inf，并选择安装(I)

完成后再按下 Windows 徽标键 + R，输入 SystemPropertiesAdvanced.exe 并回车
在“系统属性”里点击环境变量(N)...，并在“系统变量(S)”里选择 Path，点击新建(N)，并将 ADB 工具的地址输入进去，最后选择“确定”（其实你可以直接解压到与系统固件同一个目录也行）

下载完成后解压系统固件到空白文件夹

如果你不需要保留数据刷机可以直接下一步

编辑 flash-all.bat
找到“fastboot -w update image”这一行，并将 -w 给删除，完成后（如果你想升级系统，但不想 OTA 可以使用这个办法）

解压后打开 PowerShell 或 CMD，输入命令进入到 BootLoader 界面

进入到 BootLoader 之后，直接运行 flash-all.bat，这样就开始刷机了

刷完后系统自动重启，在 CMD 内按任意键即可退出

Root

打开固件里的 image-...-....zip，并将里面的 boot.img 解压出来

然后再下载 Magisk Canary
下载后使用 ADB 命令安装到 Pixel

安装完成后在手机设置搜索“USB 偏好设置”，并将“USB 的用途”改为 文件传输/Android Auto

完成后重新将手机连接到电脑，并将 boot.img 复制到手机的 Downloads 文件夹内

打开手机上的 Masigk，点击“安装” – “选择并修补一个文件”，并选择刚刚复制进去的 boot.img 文件，点击“开始→”即可修补文件

完成后在电脑上将修补好的 magisk_patched.img 复制出来

然后打开 CMD，输入命令

到第二条命令时不要着急按下回车，并注意 boot 的后面有空格，然后将 magisk_patched.img 拖进 CMD 窗口，这样就会自动补全路径了，然后再按下回车

完成后系统会重启，完成后推荐再安装一次
打开 Magisk – “安装” – “直接按照（推荐” – “开始→”，完成后点击“重启”即可

Magisk Root 之后无法正常使用系统内的 OTA，如果有需要 OTA 可以暂时先还原 Root，更新完后再 Root
亦或者是使用前文提到的去除 flash-all.cmd 中的 -w 参数来更新系统
