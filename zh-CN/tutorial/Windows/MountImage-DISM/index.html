疑难解答

如果本主题中的 DISM 命令失败，请尝试以下做法：

  1. 确保使用随 Windows ADK 一起安装的 Windows 10/11 版 DISM

  2. 不要将映像装载到受保护的文件夹，例如 User\Documents 文件夹

  3. 如果 DISM 进程中断，请考虑暂时断开网络连接并禁用病毒防护

  4. 如果 DISM 进程中断，请考虑改为从 Windows 预安装环境 (WinPE) 运行命令
